## Шаг 7: Установка SSL сертификата

1. В DirectAdmin найдите раздел **"SSL Certificates"** или **"Let's Encrypt"**
2. Выберите домен `myhelper.uz`
3. Нажмите **"Install SSL Certificate"** или **"Enable Let's Encrypt"**
4. Добавьте домены:
   - ✅ myhelper.uz
   - ✅ www.myhelper.uz
5. Нажмите **"Save"** или **"Install"**
6. Дождитесь установки сертификата (обычно 1-2 минуты)

---

## Шаг 8: Настройка редиректа HTTP → HTTPS (опционально)

### Через .htaccess файл:

1. В File Manager создайте файл `.htaccess` в директории `public_html`
2. Добавьте в него следующий код:

```apache
# Редирект HTTP на HTTPS
RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

# Редирект www на без www (опционально)
RewriteCond %{HTTP_HOST} ^www\.(.*)$ [NC]
RewriteRule ^(.*)$ https://%1/$1 [R=301,L]
```

3. Сохраните файл

## Возможные проблемы и решения

### Проблема 1: Сайт не открывается

**Решение:**
- Проверьте, что файлы загружены в правильную директорию
- Убедитесь, что файл называется `index.html` (с маленькой буквы)
- Проверьте, что домен направлен на правильный сервер

### Проблема 2: Изображения не отображаются

**Решение:**
- Убедитесь, что папка `images` загружена
- Проверьте права доступа к файлам (должно быть 644 для файлов, 755 для папок)
- Откройте консоль браузера (F12) и проверьте ошибки

### Проблема 3: SSL не работает

**Решение:**
- Подождите 5-10 минут после установки сертификата
- Проверьте, что домен действительно направлен на этот сервер
- Попробуйте переустановить сертификат

### Проблема 4: Стили не применяются

**Решение:**
- Очистите кэш браузера (Ctrl+Shift+Del)
- Проверьте, что файл `styles.css` загружен
- Откройте консоль браузера и проверьте ошибки загрузки
